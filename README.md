# Employee Database Management System (C)

A console-based Employee Database Management System developed in C using file handling.

## Features
- Add employee
- View all employees
- Search by ID
- Update employee details
- Delete employee record

## Technologies Used
- C Language
- File Handling
- GCC Compiler

## How to Run
1. Compile the code:
   gcc employee_db.c -o employee_db
2. Run:
   ./employee_db
